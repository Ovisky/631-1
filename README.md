# Flask 博客平台

这是一个基于 Python Flask 构建的轻量级博客平台，已支持：

- 博客撰写与展示
- Markdown 渲染
- 评论功能（带持久化）
- 删除文章和评论
- 基于 Render 的在线部署

访问地址 👉 [https://six31-1-1.onrender.com/](https://six31-1-1.onrender.com/)

## 技术栈

- Python + Flask
- Jinja2 模板
- JSON 文件持久化
- Markdown 渲染支持
- PicoCSS 美化前端界面
- Render 云平台部署
